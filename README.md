# Chess Unified - Admin & Operations Guide

Welcome to the new Chess Unified platform. This guide explains how to manage your website using the built-in Admin Panel.

## 🚀 Admin Access
- **Portal URL**: [http://localhost:3000/admin](http://localhost:3000/admin)
- **Password**: `admin123` *(Defaul password established during modernization)*

## 🎨 Design System
The website is built with a strictly **Blue & White Premium Theme**.
- **Pure Light Mode**: Dark mode has been completely removed as requested.
- **Color Palette**: Primary Blue (`#0062ff`), Clean White, and soft Grays.

## 🛠 Features in Admin Panel is
1. **Brand Identity**: Upload your academy's logo here.
2. **Hero Section**: Update the main headline and sub-text instantly.
3. **Location Images**: Upload multiple pictures of your address/academy location.
4. **Visibility Control**: Toggle entire sections (Programs, Gallery, etc.) on or off.

## 📁 Development
Run the local server to see changes:
```bash
npm run dev
```
